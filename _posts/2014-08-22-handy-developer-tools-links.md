There are a lot of tools out there and others have posted lists that are handy. I’m going to put my list of common tools here and try to keep it updated every now and then for my own use as well as anyone else who may like to reference it. Feel free to send me more to add to the list.

##### Helpful Blogs

  * [The Morning Brew (Chris Alcock)](http://blog.cwa.me.uk/ "The Morning Brew \(Chris Alcock\)")



##### Code & Text Editors

  * [Visual Studio](http://www.visualstudio.com/ "Visual Studio")
  * [Sublime](http://www.sublimetext.com/ "Sublime")



##### Visual Studio Extensions

  * [Web Essentials](http://vswebessentials.com/ "Visual Studio Web Essentials")
  * [SideWaffle](http://sidewaffle.com/ "Sidewaffle")
  * [StyleCop](http://stylecop.codeplex.com/ "StyleCop")
  * [ReSharper](http://www.jetbrains.com/resharper/ "ReSharper")



##### General Utilities

  * [Beyond Compare](http://www.scootersoftware.com/index.php "Beyond Compare")
  * [7-Zip](http://7-zip.org/ "7-Zip")



##### Updaters

  * [Microsoft(r) Web Platform Installer](http://www.microsoft.com/web/downloads/platform.aspx "Microsoft\(r\) Web Platform Installer")
  * [Chocolatey](http://chocolatey.org/ "Chocolatey")

  * [Epoch Converter](http://www.epochconverter.com/ "Epoch Converter")
  * [JSON Lint](http://jsonlint.com/ "JSON, or JavaScript Object Notation, is an open standard format that uses human-readable text to transmit data objects consisting of attribute–value pairs. It is used primarily to transmit data between a server and web application, as an alternative to XML.")
  * [Base64Decode](http://www.base64decode.org/ "Base64Decode")
