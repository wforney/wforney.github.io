There are a lot of tools out there and others have posted lists that are handy. I’m going to put my list of common tools here and try to keep it updated every now and then for my own use as well as anyone else who may like to reference it. Feel free to send me more to add to the list.

##### Helpful Blogs

  * [The Morning Brew (Chris Alcock)](http://blog.cwa.me.uk/)

##### Code & Text Editors

  * [Visual Studio](http://www.visualstudio.com/)
  * [Sublime Text](http://www.sublimetext.com/)

##### Visual Studio Extensions

  * [Visual Studio Web Essentials](http://vswebessentials.com/)
  * [Sidewaffle](http://sidewaffle.com/)
  * [StyleCop](http://stylecop.codeplex.com/)
  * [ReSharper](http://www.jetbrains.com/resharper/)

##### General Utilities

  * [Beyond Compare](http://www.scootersoftware.com/index.php)
  * [7-Zip](http://7-zip.org/)

##### Updaters

  * [Microsoft(r) Web Platform Installer](http://www.microsoft.com/web/downloads/platform.aspx "Microsoft\(r\) Web Platform Installer")
  * [Chocolatey](http://chocolatey.org/)

##### Web Sites

  * [Epoch Converter](http://www.epochconverter.com/)
  * [JSONLint](http://jsonlint.com/)
  * [Base64Decode](http://www.base64decode.org/)
